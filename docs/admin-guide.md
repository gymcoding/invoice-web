## 🔐 로그인

### 로그인 방법

1. 브라우저에서 `/admin-login` 페이지 접속
2. 관리자 비밀번호 입력
3. **로그인** 버튼 클릭
4. 성공 시 관리자 대시보드로 자동 이동

### 보안 정책

- 로그인 실패 5회 시 계정이 60초간 잠김
- 세션은 브라우저를 닫으면 자동 만료
- 로그아웃: 우측 상단 **로그아웃** 버튼 클릭

## 📊 견적서 목록 관리

### 견적서 조회

1. 좌측 사이드바에서 **견적서 목록** 클릭
2. 모든 견적서가 테이블 형식으로 표시됨

### 표시 정보

- **견적서 번호**: 고유 식별자 (예: INV-2025-001)
- **클라이언트명**: 견적서를 받는 고객사
- **발행일**: 견적서 발행 날짜
- **유효기간**: 견적서 유효 만료일
- **총액**: 견적 총 금액
- **상태**: 대기/승인/거절
- **링크**: 견적서 공유 URL
- **작업**: 견적서 미리보기

### 검색 기능

1. 상단 검색창에 키워드 입력
2. 검색 대상:
   - 클라이언트명
   - 견적서 번호

### 필터링

1. **필터** 버튼 클릭
2. 필터 옵션:
   - **상태별**: 대기, 승인, 거절
   - **날짜 범위**: 발행일 기준 기간 설정

### 정렬

- **발행일**: 클릭하여 오름차순/내림차순 정렬
- **총액**: 클릭하여 금액 기준 정렬

## 🔗 링크 관리

### 링크 표시

각 견적서의 **링크** 컬럼에서 고유 URL 확인:

- 형식: `https://yourdomain.com/invoice/[견적서ID]`
- URL 클릭 시 새 탭에서 견적서 페이지 열림

### 원클릭 링크 복사

1. 견적서 행의 **복사** 아이콘(📋) 클릭
2. 링크가 클립보드에 자동 복사
3. "링크가 복사되었습니다" 토스트 알림 표시
4. 복사 아이콘이 체크 표시(✓)로 변경 (2초간)

### 링크 공유

1. 견적서 행의 **공유** 아이콘(🔗) 클릭
2. 공유 옵션 선택:
   - **이메일로 공유**: 기본 메일 앱 실행
   - **텔레그램으로 공유**: 텔레그램 공유 페이지 열림
   - **링크 복사**: 클립보드에 복사

## 🌙 다크모드

### 테마 전환

1. 우측 상단 **테마 전환** 버튼 클릭
2. 테마 옵션 선택:
   - **라이트**: 밝은 테마
   - **다크**: 어두운 테마
   - **시스템**: 시스템 설정 따름

### 특징

- 선택한 테마는 브라우저에 자동 저장
- 모든 관리자 페이지에 적용
- 다크모드에서 눈의 피로 감소

## ❓ 문제 해결

### 로그인 실패

**증상**: 비밀번호를 입력해도 로그인되지 않음

**해결방법**:

1. 비밀번호 정확성 확인 (대소문자 구분)
2. 5회 실패 시 60초 대기 후 재시도
3. 문제 지속 시 관리자에게 문의

### 견적서가 표시되지 않음

**증상**: 견적서 목록이 비어있거나 로딩되지 않음

**원인**:

- Notion 데이터베이스 연결 오류
- 환경 변수 설정 누락

**해결방법**:

1. Notion Integration 권한 확인
2. 환경 변수 확인:
   - `NOTION_API_KEY`
   - `NOTION_DATABASE_ID`

### 링크 복사가 작동하지 않음

**증상**: 복사 버튼 클릭 시 아무 반응 없음

**원인**:

- 브라우저 클립보드 권한 차단

**해결방법**:

1. 브라우저 설정에서 클립보드 권한 허용
2. HTTPS 연결 확인 (로컬: http://localhost는 허용)

### 페이지 로딩이 느림

**증상**: 견적서 목록이 로딩되는데 시간이 오래 걸림

**원인**:

- Notion API 응답 지연
- 대량의 견적서 데이터

**해결방법**:

- 정상 동작: 첫 로딩 후 캐싱으로 빨라짐
- 60초 후 캐시 갱신됨

## 💡 팁

### 효율적인 견적서 관리

1. **검색 활용**: 클라이언트명으로 빠른 검색
2. **필터 조합**: 상태 + 날짜 범위로 정밀 검색
3. **링크 공유**: 이메일 공유로 클라이언트에게 빠른 전달
4. **다크모드**: 야간 작업 시 눈의 피로 감소

### 브라우저 단축키

- `Ctrl/Cmd + K`: 검색창 포커스
- `Esc`: 모달/드롭다운 닫기
